## 📌 Overview

The Smart Compound Management System is a Java Object-Oriented Programming (OOP) project designed to simulate and manage daily operations inside a residential compound.

The system improves compound efficiency by allowing residents to:
- Request services
- Report issues
- Use QR-based entry/exit tracking

Staff members can:
- Manage and monitor all activities
- Track resident movement
- Handle service requests and bus bookings

It also includes a transportation module that allows residents to book compound buses.

---

## 🚀 Features

### 👨‍👩‍👧 Residents
- Submit service requests (maintenance, support, etc.)
- Report issues inside the compound
- QR code-based entry and exit system
- View personal access history
- Book compound bus transportation

---

### 🧑‍💼 Staff / Admin
- Manage resident requests and reports
- Monitor entry and exit logs
- Track resident movement inside the compound
- Manage bus bookings
- Approve or reject service requests

---

## 🚌 Transportation Module

Residents can book seats on compound buses to designated stations, making transportation easier and more organized.

---

## 🔐 Security System

- QR-based entry and exit tracking
- Role-based structure (Resident / Staff)
- Logging of all movements and activities
